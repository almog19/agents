# agents
## types of agents
### ReAct(Reason + Act) agents
לולאה:
1) חושבים על קלט המשתמש
2) מחליטים באיזה כלי להשתמש
3) משתמש בכלי אם צריך
4) מהרהר על התוצאה
5) חוזר על הלולאה אם צריך
שימוש:
- שימוש בכלים דינאמים
- קבלת החלטות עתידיות לאחר צעד
- תעוד נקי של דרך ההיגיון
### chain of though agents
לא מבצאים פעולות מיידיות אלה, הם מסבירים את ההיגיון שלהם צעד אחר צעד לפני סיפוק תוצאה סופית
שימוש:
- צריך היגיון שאפשר לקטוע באמצע
- לא צריך כלים, צריך לוגיקה טובה
- המשימה מסובכת
